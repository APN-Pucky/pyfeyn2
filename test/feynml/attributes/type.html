type

Format: String
Elements: Leg, Propagator
Implementation: pyfeyn2.feynmandiagram.PDG

The type indicates the style of the line used to draw. Typically this can be inferred from the PDGID of the particle, but it is possible to override this by setting the type explicitly. The following types are available:
